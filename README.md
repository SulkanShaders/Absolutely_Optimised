## 💡 Overview

**Absolutely Optimised (VULKAN Edition)** is a next-gen performance modpack built to push Minecraft to its absolute limits—**without changing what makes it Minecraft**.

Every mod here has a purpose:
- Reduce lag 🧠

## 🧩 Core Mods Breakdown

### 🔥 Sulkan (Vulkan Engine)
- Brings **Vulkan-based rendering** into Minecraft  
- Dramatically improves **frame consistency & GPU efficiency**  
- Reduces CPU bottlenecks compared to traditional pipelines  

#### 🎨 Advanced Graphics Features
- 🌈 **Shader Support** — High-quality shaders with better performance than OpenGL pipelines  
- 🔦 **Dynamic Lighting** — Real-time light emission from held or moving light sources  
- 🌑 **Ambient Occlusion (AO)** — Improved depth, shadows, and visual realism  
- ⚡ **DLSS Support** *(NVIDIA)* — AI upscaling for higher FPS with minimal quality loss  
- 🧠 **XeSS Support** *(Intel)* — Smart upscaling for broader GPU compatibility  
- 🚀 Better frame pacing, reduced stutter, and smoother visuals overall  

🔗 https://sulkan.prenxy.app/
